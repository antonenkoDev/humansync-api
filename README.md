## Description

HumanSync: HR Managament Service

NestJS based API

## Installation

```bash
$ npm install
```

## Database

Create 1 Postgres Databases: 'admin', 'dev'

run migration scripts:

```bash
npm run migration:admin:run
npm run migration:dev:run
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
